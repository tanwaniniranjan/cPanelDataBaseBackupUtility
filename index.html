cPanel Database Backup Utility

cPanel Database Backup Utility is a simple shell script that runs in your terminal.

sh cPanelDataBaseBackupUtility.sh

Follow the wizard which will create a folder called backup on you desktop and download all MySQL dumps into a folder with the date
